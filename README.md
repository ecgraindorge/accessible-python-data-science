# Python Foundations to Predictive Modeling

This beginner-friendly notebook introduces Python programming for students who are new to coding and data science.

The notebook starts with foundational Python concepts, including variables, data types, lists, dictionaries, conditionals, loops, and functions. It then gradually moves into pandas, data cleaning, visualization, and a first predictive modeling workflow.

## Audience

This resource is designed for students with little or no prior programming experience. It may be useful as a pre-course resource or introductory coding workshop.

## Learning Goals

By the end of this notebook, students will be able to:

- Write basic Python code
- Use lists and dictionaries to organize information
- Use conditionals and loops to control program flow
- Write simple functions
- Load and explore tabular data with pandas
- Clean and prepare data for analysis
- Create basic visualizations
- Build and interpret a first predictive model

## How to Use This Notebook

You can download the notebook and run it in Jupyter Notebook, JupyterLab, Anaconda, or Google Colab.

No prior coding experience is required.
